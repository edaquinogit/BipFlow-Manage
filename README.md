**BipFlow-Manage** is a high-performance inventory management ecosystem designed for scalability and data-driven decision making. By bridging the gap between a robust **NestJS 11** backend and a powerful **Python** data engine, this project delivers a professional solution for modern stock control and process automation.

---

## üíé Key Features

- **Scalable REST API**: Built with NestJS 11, featuring modular architecture for long-term maintainability.
- **Data Intelligence Engine**: Python-driven core for complex data processing and inventory automation.
- **Automated Documentation**: Full API coverage with Swagger/OpenAPI, allowing real-time testing.
- **Enterprise-Grade Validation**: Global Pipes and Class-Validator integration to ensure data integrity and security.
- **Unified Database**: Shared SQLite architecture between Node.js and Python environments.
